# Shareable Achievement Version Notes

This version fixes the current development state as a shareable PoC achievement package.

## Main changes

1. `README.md` was completely rewritten.
2. The dashboard was redesigned so that the live video and live AI overlay are the primary visual focus.
3. Long tables were moved into fixed-height scrollable panels.
4. Forecast, recommendation, incident, detection, analytics, and system tables no longer dominate the main screen.
5. The UI includes clearer prototype disclaimers and limitations.

## Files changed

- `README.md`
- `frontend/static/index.html`
- `frontend/static/style.css`

No backend API changes were required for this UI/documentation consolidation.
